## ğŸ› Reporting Issues

### What to Report
- **Crashes**: App closes unexpectedly
- **Freezes**: App becomes unresponsive
- **Errors**: Error messages or warnings
- **Missing Features**: Expected features not working
- **Bad Responses**: MOTTO gives wrong/unhelpful answers

### How to Report

**Include:**
1. **What happened**: Describe the issue
2. **When**: When did it occur?
3. **Device**: iPhone model / Android device
4. **OS Version**: iOS 15.0 / Android 12, etc.
5. **Steps to Reproduce**: How to make it happen again

**Example:**
```
Issue: App crashes when I send a message

Device: iPhone 13 Pro
OS: iOS 16.0
When: Every time I tap send

Steps:
1. Open app
2. Type "Hello"
3. Tap send
4. App crashes
```
